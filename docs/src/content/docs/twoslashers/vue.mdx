Included in Expressive code twoslash, [Twoslash Vue](https://twoslash.netlify.app/packages/vue) is a Twoslash integration that added support for Twoslash to handle Vue Single File Component files.

<Aside>
Usage of this integration requires installing the `vue` package as a dependency in your project, as it's a peer dependency of Twoslash-Vue. You can install it using npm, pnpm, or yarn:

```bash
npm install vue
```
</Aside>

This is an example taken from the Twoslash Vue documentation website.

<Tabs>

<TabItem label="Output">

```vue twoslash
<script setup>
import { onMounted, ref } from 'vue'
//                   ^?

// Reactive state.
const count = ref(0)

// Functions that mutate state and trigger updates.
function increment() {
  count.value++
}

// Lifecycle hooks.
onMounted(() => {
  console.log(`The initial count is ${count.value}.`)
})
</script>

<template>
  <button @click="increment">
    Count is: {{ count }}
  </button>
</template>
```
</TabItem>
